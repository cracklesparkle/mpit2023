# ТУТиТАМ - турнавигатор
Проект команды WEBM на МПИТ-2023.

## Описание
Мобильное приложение агрегатор досуговых активностей (например туры, маршруты, отдых) на Flutter.

## Стек
Flutter + Firebase + Supabase
